Razer Nabu Open SDK  0.3.0
The Razer Nabu Open SDK, or Open SDK, allows access to fitness data, fitness history data, sleep data and user profile from our Razer server. Aside from these, the Open SDK allows you to send custom notification from your app to the band.

Detailed Description

The NabuDataManager class is responsible for all interactions between the NabuSDK and the NabuAPI.

Method Documentation

- (NSURL *) authorizationURLForAppID: (NSString *)  appId
andScope: (NSString *)  appScope
withAppURISchemeCallback: (NSString *)  appURICallback 

This method will construct a URL using the formatted information from client app Id, scope and URL scheme callback.

Parameters
appIdYour client's App Id - This is your App Key from Nabu Developer Portal.
appScopeYour client's Scope upon registration
appURICallbackYour client's custom defined URL scheme for app to app communication. Must be the root scheme and not like this format(i.e., RZOrg://page=1)
Returns
A constructed URL formed data that will be called to request authorization in order to get access to certain Nabu information from the cloud.
  • Example: 
      NSURL *authorizationURL = [[RzDataManager sharedDataManager] authorizationURLForAppID:@"j293kalc49al0k1msmz9ks0ag8cdd0a0ca"
                                                                               andScope:@"fitness"
                                                               withAppURISchemeCallback:@"RZOrg"

  [[UIApplication sharedApplication] openURL:authorizationURL];
